THE FIRST THE LAST

The First the Last is a website advertising The First The Last design portfolio.
It is highly interactive, displaying elements of HTML, CSS and JavaScript - featuring elements such as custom cursors and interative backgrounds.
Along with the main aesthetic being very modern, there is an integration of uncommon elements - such as buttons being a part of the actual text.
Choices like these allow for a natural progression throughout the page - allowing the user to discover what the site has to offer rather than being in ours faces.
